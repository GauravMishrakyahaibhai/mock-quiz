# 🎯 Mock Quiz App

🚀 A modern, interactive quiz web application built using React and Vite.
Challenge yourself with multiple-choice questions and track your performance in real-time!

---

## 🌍 Live Demo

🔗 **Try it here:**
https://mock-quiz-zmmh.vercel.app

---

## ✨ Features

* 🧠 Engaging multiple-choice quiz system
* ⚡ Instant answer validation
* 📊 Real-time score tracking
* 📄 Multi-page navigation (Home, Quiz, Tests, Score)
* 📱 Fully responsive design
* 🎨 Clean and minimal UI

---

## 🛠️ Tech Stack

| Technology      | Purpose         |
| --------------- | --------------- |
| ⚛️ React        | Frontend UI     |
| ⚡ Vite          | Fast build tool |
| 🎨 Tailwind CSS | Styling         |
| 📦 Node.js      | Runtime         |

## 🚀 Getting Started

### 1️⃣ Clone the repository

```
git clone https://github.com/GauravMishrakyahaibhai/mock-quiz.git
```

### 2️⃣ Navigate to project folder

```
cd mock-quiz
```

### 3️⃣ Install dependencies

```
npm install
```

### 4️⃣ Start development server

```
npm run dev
```
